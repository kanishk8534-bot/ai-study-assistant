# Personal Tutor AI

Personal Tutor AI is an AI-powered study assistant built using Python.

## Features
- Answers study questions
- Generates quizzes
- Creates study plans
- Helps students learn efficiently

## Technology Used
- Python
- Streamlit
- AI APIs

## How to Run

1. Install Python
2. Install required packages
3. Run:

python assistant.py
